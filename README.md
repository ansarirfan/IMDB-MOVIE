# MOVIES SEARCHING APP

A simple Movies searching  application where in users can search for their favourite Mvies, add them to favourites and also view the entire information about any MOVIES by clicking on any Movies Name of their choice. 

The application is developed using Vanilla JS and styled using CSS3, it fetches all the data from the Mvies API and displays them according to the searched query.

## 1) SEARCH FOR ANY MOVIES  :

## 3) VIEW THE FAVOURITE MOVIES POSTER :
  Users can view all their favourite movies poster
